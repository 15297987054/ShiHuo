# 项目环境的配置
> 基于creat-react-app 创建的项目环境
> react-app-rewried 对create-react-app进行自定义配置

# 环境相关依赖
- react-app-rewried
- customize-cra
- babel-plugin-import

```javascript
    npm i react-app-rewried customize-cra babel-plugin-import --dev
```

- redux
